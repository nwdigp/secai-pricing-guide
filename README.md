1. **打开相机，对准画面**——实时捕捉环境色彩，自动生成色卡，适合户外旅拍、街头观察配色。

## 价格怎么算才不亏？

价格顾虑基本是绕不开的一环。换个算法看，永久版本质上是把多年的使用成本摊薄到一次性支付里，如果你打算长期用，分摊到每个月其实没多少钱；但如果只是临时需要做一两个项目的配色，一月版显然更划算，不用为用不到的时长多花钱。

数码荔枝这边的定价逻辑，官方说法是向开发商批量采购降低单套成本，所以通常比软件官网价格更友好，同时保留正版授权和售后支持。这一点在页面的购买须知里也写得比较清楚：所有商品均为官方授权激活码或授权文件，不是破解或共享账号。

## 常见问题

**色采的高级版需要单独购买每个平台吗？**
授权方案区分单平台和双平台版本，购买时需要按自己的使用设备选择对应规格，双平台版本可在 iOS 和 Android 之间同步使用。

**购买的授权码可以激活官网下载的色采吗？**
可以。正版授权码与官网下载版本完全兼容，激活流程一致，不存在渠道差异导致无法激活的问题。

**色采有破解版吗，能不能用？**
市面上确实存在破解版本，但盗版软件常常夹带木马或后门程序，存在隐私和设备安全风险，不建议为了省钱去用，免费版的功能已经能满足基础需求。

**买错版本或者激活失败怎么办？**
数字软件属于特殊商品，一般不支持无理由退款，但如果遇到激活异常、买错规格等情况，且授权可以回收，是可以申请售后处理的，具体走平台的售后政策流程。

**换手机之后还能继续用之前买的授权吗？**
可以，但需要先在旧设备上登出账号完成授权转移，之后在新设备登录同一账号即可继续使用。
